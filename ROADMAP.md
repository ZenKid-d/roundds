# Roundds — планы (roadmap)

## Готово ✅
- Фоновое воспроизведение через `audio_service` (уведомление, lock-screen).
- Кросс-источниковый fallback (недоступный трек играем из YouTube).
- SoundCloud по OAuth-токену подписки (полные Go+ треки).
- Реальные **shuffle / repeat** (off/all/one).
- **«Избранное»** (лайк трека) + вкладка в Медиатеке.
- **Текст песен** (синхронный/обычный, lrclib.net) — экран с автоскроллом.
- **Скачивание/оффлайн** — загрузка треков, проигрывание из файла, вкладка «Загрузки».
- **Предзагрузка** следующего трека (быстрое переключение).
- **Эквалайзер** (по полосам) + **таймер сна** (15/30/45/60 мин) — в меню «•••» плеера.
- **Страница артиста** (тап по имени артиста → его треки).
- **Статистика прослушиваний** (топ треков/артистов) — в Настройках.
- **Импорт плейлистов**: YouTube по ссылке + Яндекс из аккаунта (по токену).
- **Настройки «Внешний вид»**: фон (AMOLED/тёмно-серый), акцент (динамический/пресеты/свой
  цвет-пипетка), вид плеера (винил/обложка), уровень анимаций, скругления углов, шрифт.
- **Рекомендации**: персональные ряды на главной («Похожее на…», «Ещё от…», «Для вас»)
  на базе истории/лайков/статистики + эндпоинты похожих (Яндекс/SoundCloud) и подбор
  по артисту. **Радио** — бесконечный микс, очередь докручивается похожими на лету.
- **Импорт лайков YouTube Music** через официальный OAuth (YouTube Data API).
- **Кроссфейд-лайт** (затухание/появление на границах) + **скорость воспроизведения**.
- **Умные плейлисты** («Часто слушаю», «Недавнее», «Только лайки») + **бэкап**
  (экспорт/импорт библиотеки в JSON).
- **Радио по артисту** (кнопка на странице артиста) и **по жанру** (чипы на главной).
- **«Новинки и чарты»** — отдельный экран с чартом по каждому сервису.
- **Караоке-режим** — полноэкранный синхронный текст поверх размытой обложки.
- **Аним-фон плеера** — живой градиент из акцента (гейтится уровнем анимаций).

## Бэклог
- Настоящий кроссфейд с наложением (нужны два плеера) — сейчас fade-лайт.
- Гэпплес; локальные файлы устройства (нужны права + скан медиатеки).
- Виджет на экран / плитка в шторке; Android Auto (нативное, нужен тест на устройстве).
- Альбомные страницы; кэш результатов поиска/ленты; прогресс скачивания в %.
- (Вне проекта) Spotify — невозможен без обхода DRM; VK — не выбран.
